# 🐈 Pan'n'Zoom

Pan'n'Zoom is a web based app, that lets you present panning and zooming within your images.

The app is purely client-based.

# ⭐️ Let me convince you
- ✅ Free and Open Source
- ✅ Offline Mode
- ✅ Desktop App
- ✅ Help Videos
- ✅ Export and Import
- ✅ Convenience Functions
- ✅ Light + Dark Mode
